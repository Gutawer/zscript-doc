# SectorEffect

A thinker which is attached to a sector and effects it in some way.

```
class SectorEffect : Thinker
{
   protected Sector m_Sector;

   Sector GetSector();
}
```

- `m_Sector`

   The sector this effect is attached to.

- `GetSector`

   Returns the sector this effect is attached to.
